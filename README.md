# PulseBoard 📊

**Predictive Business Intelligence for SMBs**

PulseBoard is a premium, AI-powered analytics dashboard designed to help small and medium-sized business owners understand their revenue data through plain-English insights and advanced forecasting.

---

## 🚀 Key Features

- **📤 Smart Data Upload**: Support for CSV and Excel files with automatic column detection.
- **🔮 AI-Powered Forecasts**: 4-week revenue predictions powered by Meta's **Prophet** model.
- **💡 Plain-English Insights**: Daily business summaries and action recommendations generated by **Google Gemini 1.5 Flash**.
- **⚠️ Anomaly Detection**: Automatic alerts for unusual revenue spikes or dips.
- **🔥 Premium UI**: Sleek, dark-themed interface built with Streamlit and modern CSS.

---

## 🛠️ Technology Stack

- **Frontend**: Streamlit
- **Forecasting**: Prophet (Meta)
- **AI/LLM**: Google Gemini 1.5 Flash
- **Database/Auth**: Firebase (Optional)
- **Data Analysis**: Pandas, NumPy, Plotly

---

## ⚙️ Setup & Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/tarakishore/pulseboard.git
   cd pulseboard
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure Environment Variables**:
   Create a `.env` file in the root directory and add your keys:
   ```env
   GEMINI_API_KEY=your_gemini_api_key
   FIREBASE_API_KEY=your_firebase_key (optional)
   ```

4. **Run the Application**:
   ```bash
   streamlit run app.py
   ```

## 📂 Project Structure

- `app.py`: Main entry point.
- `pages/`: Multi-page Streamlit application structure.
- `ai/`: Gemini integration and prompt engineering.
- `ml/`: Forecasting, anomaly detection, and business metrics.
- `core/`: Config, auth, database, and session management.
- `ui/`: Custom styles and reusable components.
